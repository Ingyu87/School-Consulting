# School Consulting Planner

찾아가는 학교 컨설팅 심층면담 현장에서 사용하는 진단 분석 및 문서 생성 웹앱입니다.

## 목표

- 학교별 사전 자가진단 CSV 업로드
- 진단 결과 시각화 및 분석 초안 생성
- 심층면담지 작성
- 연수 모듈/일정 구성 및 PDF 공식 기준 검증
- 운영계획서 작성
- `심층면담지.docx`, `운영계획서.docx` 다운로드

## 실행

```bash
npm install
npm run dev        # UI만 확인 (AI·녹음 기능은 /api 함수가 없어 동작하지 않음)
```

AI 초안 생성과 면담 녹음 전사는 Vercel Functions(`api/`)를 사용하므로,
로컬에서 전체 기능을 확인하려면 `vercel dev`로 실행해야 합니다.

```bash
npm install -g vercel
vercel dev         # GEMINI_API_KEY, NEIS_API_KEY 환경변수 필요
```

환경변수:

- `GEMINI_API_KEY`: AI 초안 생성 및 면담 분석용 Gemini API 키
- `NEIS_API_KEY`: 나이스 교육정보 개방 포털 학교기본정보 조회용 인증키

프로덕션 빌드 확인:

```bash
npm run build
```

## 현재 구현

- `서울고일초_사전 자가진단 분석.csv` 형식의 CSV 업로드 및 마커 기반 파싱
- 과정별 평균 점수·인프라 문항 응답 분포·서술형 응답 시각화, 우선 확인/강점 영역 자동 요약
- 심층면담지.pdf 전 섹션 입력: 필수 안내 체크, 운영 개요(코디네이터·참여 교원 명단), 학교 일반사항·인프라 현황, 참여 목표 체크, 디지털 친화도 진단, 기타 고려사항 4구분, 핵심 요약
- 운영계획서.pdf 전 섹션 입력: 강점·도전과제, 1·2차 면담 요약, 이슈→목표 도출, 과정별 세부 프로그램·기대효과
- 0~7과정 카드에서 차시/방식/일정/시간/장소/인원/회차/중점 도구/운영 주제/프로그램명 입력
- 면담 녹음: 3분 단위 구간 분할 → 구간별 자동 전사 → 종료 시 면담 항목 AI 자동 작성
- Gemini 기반 AI 초안(진단 시사점, 강점·과제, 이슈→목표, 프로그램 초안 등)
- 나이스 교육정보 개방 포털 학교기본정보 API로 학교 주소·학교급·설립구분 자동 반영
- IndexedDB 기반 자동저장, 백업 JSON 저장/복원(형식 검증 포함)
- PDF 공식 기준 검증: 12차시, 필수과정, 선택 3개, 모듈1~3 합계 5차시, 교원 대상 모듈 단위 2차시, 온라인 30%, 필수 모듈 오프라인, 운영 순서 안내
- `심층면담지.docx`, `운영계획서.docx`를 PDF 양식 섹션 순서 그대로 생성

## 배포 방향

Vercel 배포를 기준으로 구현합니다.

1차 구현은 Vite + React + TypeScript 기반 정적 앱으로 시작합니다.
DOCX 품질이 더 필요할 경우 Vercel Functions 기반 문서 생성 API를 추가합니다.
